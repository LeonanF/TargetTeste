1) Observe o trecho de código abaixo:

int INDICE = 13, SOMA = 0, K = 0;

enquanto K < INDICE faça

{

K = K + 1;

SOMA = SOMA + K;

}

imprimir(SOMA);



Ao final do processamento, qual será o valor da variável SOMA?

Resposta: Soma será igual a 91. O código soma os primeiros N números inteiros, então a fórmula é: n*(n+1)/2, onde n = indice
